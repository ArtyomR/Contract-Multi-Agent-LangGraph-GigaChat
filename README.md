# Мульти-агентная система проверки договоров (прототип)
Мы создадим прототип системы на базе LangGraph и GigaChat. 
Система состоит из двух специализированных агентов:
1. **Агент-юрист**: ищет правовые риски, проверяет соответствие законодательству и корректность формулировок.
2. **Агент-финансист**: анализирует условия оплат, штрафов, пеней и финансовые риски.

Код сгенерирован 2мя LLM
1. Alice AI (Яндекс). Агенты работают в параллель. Третий агент составляет общий отчёт
2. Qwen (Alibaba Cloud). Агенты работают последовательно: сначала юрист, затем финансист (он может учесть выводы юриста).
